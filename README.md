# Vende+ Clean

Versión limpia del catálogo inteligente Vende+.

## Ejecutar en local

```powershell
cd C:\Users\Windows\Downloads\vendeplus-clean
npm.cmd config set registry https://registry.npmjs.org/
npm.cmd install --registry=https://registry.npmjs.org/
npm.cmd run dev -- --hostname 0.0.0.0
```

Abrir:

- http://localhost:3000
- http://localhost:3000/don-aniello
- http://localhost:3000/china-twon
- http://localhost:3000/bizcochos-ascoli
